# bush
bush is a minimal shell, that barely works.

dont expect much out of it, it may lack some important features and it may have bugs.

# how to use
```
+-------------------------------------------------------------------+
| -> bush           [Shell script]...                               |
| -> |              Pipe the input before it to the command after   |
+-------------------------------------------------------------------+
```

# installing / compiling
run 
```
# make
```
to compile

run 
```
# make install
```
to install
